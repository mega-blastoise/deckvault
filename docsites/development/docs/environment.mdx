# Environment Variables

## Web (`apps/web`)

| Variable | Required | Default | Description |
|----------|----------|---------|-------------|
| `PORT` | No | `3000` | HTTP port |
| `API_URL` | Yes | â€” | REST API base URL including `/api/v1` |
| `REST_API_URL` | Yes | â€” | REST API base URL without path prefix |
| `GRAPHQL_API_URL` | Yes | â€” | GraphQL API base URL |

**Local dev (`apps/web/.env`):**
```
API_URL=http://localhost:3001/api/v1
REST_API_URL=http://localhost:3001
GRAPHQL_API_URL=http://localhost:3002
```

**Production (injected by `docker-compose.prod.yml`):**
```
API_URL=http://rest-api:3001/api/v1
REST_API_URL=http://rest-api:3001
GRAPHQL_API_URL=http://graphql-api:3002
```

|----------|----------|---------|-------------|

## Production Secrets

In production, secrets are stored in `/opt/deckvault/.env.prod` on the Spark (chmod 600, owned by `deckvault`). Referenced via `env_file` in `docker-compose.prod.yml`. Never committed to the repository.

<Callout variant="warning">
  Never commit `.env`, `.env.prod`, or any file containing real credentials. The `.gitignore` already excludes `.env*` files.
</Callout>
