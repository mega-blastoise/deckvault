# Changelog

All notable changes to the `@johto-ai/*` packages will be documented in this file.

This changelog is updated automatically by `bun scripts/changes.ts release`.

---

## @johto-ai/mcp-server v0.1.0 (2026-05-18)

- First release of @johto-ai/mcp-server — MCP server binary exposing Pokemon TCG data and deck analysis tools to the johto CLI.

## @johto-ai/card-data v0.1.0 (2026-05-18)

- First release of @johto-ai/card-data — prebuilt SQLite database containing Pokemon TCG card and set data for use by the johto CLI.

## @johto-ai/cli v0.1.0 (2026-05-18)

- First release of @johto-ai/cli — competitive Pokemon TCG deck refinement CLI with Anthropic agent loop and MCP server integration.
